# Global Warming Quiz (English + Hindi)

## Setup Instructions

1. Extract the ZIP file on your laptop.
2. Open `index.html` in your browser (Chrome, Firefox, Edge).
3. The quiz will load with 40 questions (20 kids + 20 teens/pre-adults).
4. Each question has 4 options and an embedded video.

## Adding Videos
- Go to free sources like [LottieFiles](https://lottiefiles.com), [Pexels](https://pexels.com), [Pixabay](https://pixabay.com), [Videvo](https://videvo.net).
- Download **cartoon-style MP4 clips** matching each question.
- Rename the videos as `q1.mp4`, `q2.mp4`, ..., `q40.mp4`.
- Place them inside the `/videos` folder.

## Example Keywords for Videos
- q1: "turn off lights animation"
- q2: "walking vs driving animation"
- q3: "tree planting animation"
- q4: "water wastage animation"
- q5: "plastic bottle recycle animation"
- ...
- q40: "carbon sequestration animation"

## Hosting on Netlify
1. Go to https://app.netlify.com/drop
